# Saffron & Oak — Restaurant Website

A fully responsive restaurant website built with pure HTML, CSS, and JavaScript.

## Features
- Responsive navigation
- Hero section with CTAs
- Menu cards with pricing
- Testimonials section
- Reservation form
- Mobile friendly

## Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
