# Saffron & Oak — Restaurant Website

A fully responsive restaurant website built with pure HTML, CSS, and JavaScript.

## Live Demo
[View Live Site](https://saffron-oak.netlify.app)

## Features
- Responsive navigation bar
- Hero section with stats and CTAs
- Menu cards with add-to-cart interaction
- About section with features
- Customer testimonials
- Reservation form with validation
- Scroll reveal animations
- Mobile friendly layout

## Tech Stack
- HTML5 (semantic structure)
- CSS3 (Flexbox, Grid, custom properties)
- Vanilla JavaScript (DOM manipulation, IntersectionObserver)

## Project Structure
```
Saffron-oak/
│
|\ index.html        # Main HTML structure
|\ style.css     # All styles
│\ main.js       # Interactions & animations
|\ README.md
```

## JavaScript Features
- Smooth scroll navigation
- Add to cart button feedback
- Reservation form validation
- Scroll reveal animations using IntersectionObserver API
